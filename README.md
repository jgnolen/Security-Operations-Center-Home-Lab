(project in-progress)

## Objective
- Create a SOC environment, simulate attacks, defend attacks, then write reports.

## Steps
1. Create VMs for the homelab
- ...

2. Configure VMs for the homelab
- This includes testing connectivity and installing updates (excluding specific vulnerable VMs).
- This step completes the homelab creation and setup.
- ...

3. Simulate an attack by using Kali VM or by using Atomic Red Team. Observe the patterns of specific attacks simulated.
- This step includes documenting as you go as the defender.
- ...

4. Harden the network and repeat attack.
- This step may include creating additional alerts and reconfigurations.
- Repeat step until the network is successfully defended.

5. Complete a report as the incident handler for each simulated attack.
- ...
